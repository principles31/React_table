# React_table
A React table character with a name and a job where a user can add and remove job to the table.
## Built With

- HTML
- CSS
- Gitflow
- JAVASCRIPT
- React
- Redux

## Getting Started
To get a local copy up and running follow these simple example steps.
### Prerequisites

- Web browser
- Code editor

### Setup


```bash
npm install
```

```bash
npm build
```

```bash
npm start
```
